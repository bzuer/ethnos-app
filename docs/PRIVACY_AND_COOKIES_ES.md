# Privacidad y Cookies

Ethnos Bibliography está diseñado para respetar tu privacidad. Esta página explica, en lenguaje claro, qué información usa la aplicación, por qué la usa y qué opciones tienes según las normas comunes de la UE y Brasil (RGPD y LGPD).

## Lo Que Usamos

### Preferencia de Idioma
Tu navegador puede guardar una preferencia de idioma para recordar tu elección. La aplicación lee esa preferencia para mostrar la interfaz en el idioma correcto. No la usamos para seguimiento.

### Tu Lista Personal
Si guardas obras en tu lista personal, la lista se almacena solo en tu propio navegador. Incluye los datos básicos necesarios para mostrar y exportar tus elementos guardados. Permanece en tu dispositivo hasta que la borres.

### Búsquedas y Solicitudes a la API
Cuando buscas o navegas listas, tus términos y filtros se envían a la API para obtener resultados. Estas solicitudes pasan por un proxy del servidor que reenvía solo la ruta y la consulta que pediste. No envía tus cookies, tu dirección IP ni los datos de tu navegador a la API.

## Lo Que No Recopilamos
- No usamos analítica ni rastreadores publicitarios.
- No hay cuentas de usuario ni perfiles.
- No usamos sessionStorage ni IndexedDB.

## Registros del Servidor
El servidor mantiene registros básicos de ejecución para mantener el servicio estable. No hay seguimiento personalizado de usuarios ni de sus búsquedas en este proyecto. Cualquier registro adicional solo existiría si se agrega mediante infraestructura externa.

## Tus Derechos y Opciones
Puedes borrar tu lista personal en cualquier momento desde la página Mi Lista o eliminando los datos del sitio en tu navegador. También puedes cambiar el idioma en tu navegador o borrar la preferencia de idioma. Bajo RGPD y LGPD, tienes derechos para acceder, corregir o eliminar datos personales que puedan estar vinculados a ti. Como esta app guarda la lista localmente en tu navegador, tú controlas esos datos.
